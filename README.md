# 🍔 Diet Tomorrow — Food Ordering System

A full-stack food ordering web application with login, menu browsing, cart system, and order processing.

---

## 🚀 Features

* 🔐 User Authentication (Login system)
* 🍱 Dynamic Menu Display
* 🔍 Search & Filter Food Items
* 🛒 Cart with Price Calculation
* 💎 Modern UI (Glassmorphism Design)
* 📊 Backend API with Express.js

---

## 🛠 Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express.js
* Database: SQLite

---

## ⚙️ Setup & Run

```bash
npm install
node server.js
```

Open:
http://localhost:5000

---

## 📌 Project Structure

* `food order system/` → Frontend files
* `server.js` → Backend
* `database.sqlite` → Database

---

## 🎯 Purpose

This project demonstrates full-stack development with database integration for a food ordering system.
